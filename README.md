# 🛒 Flipkart Clone - React + Vite

A highly readable, beginner-friendly Flipkart Clone built with React and Vite. This project focuses on clean code, simple state management, and modern UI/UX principles.

## ✨ Features

- **🏠 Home Page**: Explore a variety of products with a clean grid layout.
- **🔍 Search**: Quickly find products using the real-time search functionality.
- **🏷️ Categories**: Browse products by category using the interactive category strip.
- **⚖️ Filtering & Sorting**: Narrow down products by price range, rating, and availability. Sort by price or rating.
- **🕐 Recently Viewed**: Keep track of products you've recently visited.
- **📄 Product Details**: View detailed information, specifications, and high-quality images for each product.
- **🛒 Shopping Cart**: Add items to your cart, update quantities, and see total costs.
- **❤️ Wishlist**: Save your favorite items for later and easily move them to the cart.
- **🌓 Dark Mode**: Toggle between light and dark themes for a personalized experience.
- **👤 User Auth**: Simplified login/logout functionality with local persistence.
- **💾 Data Persistence**: Your cart, wishlist, and user settings are saved in `localStorage`.
- **🔔 Toast Notifications**: Real-time feedback for user actions (e.g., adding to cart).

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Routing**: [React Router DOM v6](https://reactrouter.com/)
- **Styling**: Vanilla CSS (Modern, Responsive, Glassmorphism)
- **State Management**: React Hooks (`useState`, `useEffect`)
- **Persistence**: Web Storage API (`localStorage`)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.0.0 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)


## 📁 Project Structure

```text
src/
├── components/   # Reusable UI components (Navbar, Toast, ProductCard, etc.)
├── pages/        # Main application pages (Home, Cart, Wishlist, etc.)
├── services/     # API or mock data services
├── styles/       # CSS modules and global styles
├── data/         # Mock product data
└── App.jsx       # Main application logic and routing
```

## 📝 Design Philosophy

- **Simplicity**: Step-by-step logic that is easy for beginners to follow.
- **Readability**: Descriptive variable naming and helpful comments.
- **Visual Excellence**: Premium feel with smooth transitions, gradients, and micro-animations.
